# readMe
GiffyGram Project for NSS

## Installation
git clone this repository to your local directory.

# Testing Instructions
1. run `serve` in root

2. run `json-server -p 8088 -w giffygram.json` in the api directory

3. navigate in browser to `localhost:5000/src`
